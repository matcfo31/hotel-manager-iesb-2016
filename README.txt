*************************** Arquivos do Sistema de Login **************************

[autenticadoATUAL] 	- Guarda o nome do usuário que fez login no sistema
[Admins.txt] 		- Guarda nome e senha do usuário no formato "nome\tsenha\n"
[Funcionarios.txt] 	- Guarda nome e senha do usuário no formato "nome\tsenha\n"
[Geral_nomes.txt] 	- Guarda apenas o nome do usuário


****************************** Arquivos de Registro ******************************* 

[nomedofuncionario.txt]	- Armazena quantidade de check-ins do funcionário
[numerodoquarto.txt]	- Armazena quantidade de check-ins nesse quarto
[nomedohospede.txt]	- Armazena quantidade de check-ins desse hóspede
[AllGuests.txt]		- Nome de todos os hóspedes que já fizeram check-in

****************************** Arquivos dos Quartos *******************************

[QuartosDesocupados.txt] 	- Guarda apenas o número dos quartos desocupados
[QuartosOcupados.txt]		- Guarda apenas o número dos quartos ocupados
[Quartos.txt]	 		- Guarda o número e capacidade dos quartos
